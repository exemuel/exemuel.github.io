This project features an interactive data visualization dashboard hosted on [Tableau Public](https://public.tableau.com/app/profile/samuel.situmeang/viz/RegionalComparisonofZillowHomeValueIndex/U_S_HousingMarketExplorerZHVI), designed to analyze and compare housing market trends across the United States. 

### Project Overview
Understanding real estate market dynamics is crucial for economic analysis. This dashboard utilizes the **Zillow Home Value Index (ZHVI)**—a smoothed, seasonally adjusted measure of the typical home value—to provide a comprehensive view of the U.S. housing market. 

The primary goal of this visualization is to allow users to interactively explore how typical home values have shifted over time across different regions, states, and metropolitan areas. 

### Key Features
* **Regional Comparisons:** Interactive maps and filters that allow users to drill down into specific geographic areas and compare home value growth rates.
* **Trend Analysis:** Time-series visualizations tracking historical ZHVI data to identify market peaks, corrections, and long-term appreciation trends.
* **Interactive Filtering:** Dynamic controls that enable users to segment the data by specific timeframes and regional boundaries.

### Tools & Technologies Used
* **Tableau:** Utilized for data blending, aggregation, and building the interactive user interface.
* **Data Source:** Zillow Home Value Index (ZHVI) public datasets.
